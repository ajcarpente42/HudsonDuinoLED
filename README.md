# HudsonDuinoLED

## Requirements

 * Arduino
 * 8x8 LED Matrix
 * Python 3
 * PySerial (http://pythonhosted.org/pyserial/)

## Usage
 * In command terminal, run: `python Python/HudsonCompanion.py <port>`
 * Optional: URL: `python Python/HudsonCompanion.py <port> <url>`
   * Where `<url>` is a Hudson remote API XML file
